Detailed Description

Class to calculate the pointing (i.e. viewing) angles (azimuth elevation) from a topocentric frame. The rotations between an inertial and body-fixed and body-fixed and tropocentric are provided to the constructor. <ember functions can be used to calculate the pointing angles from the ground station to a vector in an inertial frame and a time.
