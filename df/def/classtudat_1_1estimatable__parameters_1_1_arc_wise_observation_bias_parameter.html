Detailed Description

Interface class for the estimation of aan arc-wise constant absolute or relative observation bias (at given link ends and observable type). Unlike most other EstimatableParameter derived classes, this class does not have direct access to the class (ConstantArcWiseObservationBias/ ConstantRelativeArcWiseObservationBias) used in the simulations for the observation bias. This is due to the fact that the ConstantArcWiseObservationBias and ConstantRelativeArcWiseObservationBias class are templated by the observable size, while this class is not.
