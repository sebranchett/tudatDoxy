Detailed Description

Class for providing settings for torque model. This class is a functional (base) class for settings of torque models that require no information in addition to their type. Classes defining settings for torque models requiring additional information must be derived from this class. Bodies exerting and undergong torque are set externally from this class. This class can be used for the easy setup of torque models (see createTorqueModels.h), but users may also chose to do so manually. (Derived) Class members are all public, for ease of access and modification.
