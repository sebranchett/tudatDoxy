Detailed Description

Base class for computing the direction of a given force/acceleration. The computation is done directly from some a priori imposed rule, possibly as a function of dependent, independent or state variables. A derived class for using the current orientation of the vehicle (as computed by some other method and retrieved from the body class) may be set using the OrientationBasedForceGuidance class.
