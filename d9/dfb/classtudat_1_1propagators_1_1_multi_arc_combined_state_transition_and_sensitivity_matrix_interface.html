std::vector< std::vector< std::pair< int, int > > > tudat::propagators::MultiArcCombinedStateTransitionAndSensitivityMatrixInterface::statePartialAdditionIndices_
private

Vector of pair providing indices of column blocks of variational equations to add to other column blocks, which is needed by the hierarchical estimation fo dynamics. The second pair entry is the start of the column block (of size 3) to where it should be copied. The first entry denotes from where it should be copied.
