Writes the points on a SingleSurfaceGeometry object to a file. Each row contains the x, y and z coordinate of the point, each next row defines a single new point.

Parameters
singleSurfaceGeometryGeometry which is to be written.
numberOfLinesDefines how many points are taken over the 1st independent variable.
numberOfPointsDefines how many points are taken over the 2nd independent variable.
filenameName of the file to which the points are written.
writeTypeDefines whether to append or write to the file given by filename, should be "a" for append and "w" for write.
isIndependentVariableInvertedBoolean flag which if set to true inverts which independent variable is treated as 1st and which as 2nd.
