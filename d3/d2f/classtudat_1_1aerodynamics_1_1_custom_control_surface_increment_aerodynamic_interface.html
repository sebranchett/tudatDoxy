Detailed Description

Control surface aerodynamic coefficient interface taking function pointers providing aerodynamics coefficient increments as a function of independent variables (doubles).

The origin of the coefficients or the nature of the independent variables is irrelevant for this class. A typical use of this class is to interface interpolated tabulated control surface increments.
