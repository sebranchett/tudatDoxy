Detailed Description

template<int ObservationSize = 1>
class tudat::observation_models::MultiTypeObservationBias< ObservationSize >

Class for combining multiple observation bias models into a single bias value. This class computes a list of biases, all based on the nominal, unbiased, observation and sums them up to form the total observation bias.
