Detailed Description

template<typename IndependentVariableType = double>
class tudat::numerical_integrators::IntegratorSettings< IndependentVariableType >

Class to define settings of numerical integrator, for instance for use in numerical integration of equations of motion/ variational equations. This class can be used for simple integrators such as fixed step RK and Euler. Integrators that require more settings to define have their own derived class (see below).

template<typename IndependentVariableType = double>
bool tudat::numerical_integrators::IntegratorSettings< IndependentVariableType >::assessPropagationTerminationConditionDuringIntegrationSubsteps_

Whether the propagation termination conditions should be evaluated during the intermediate updates performed by the integrator to compute the quantities necessary to integrate the state to a new epoch. The default value is false, so the propagation termination condition is only checked at the end of each integration step.
