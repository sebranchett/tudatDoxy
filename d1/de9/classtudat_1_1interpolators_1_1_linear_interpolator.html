Detailed Description

template<typename IndependentVariableType, typename DependentVariableType>
class tudat::interpolators::LinearInterpolator< IndependentVariableType, DependentVariableType >

This class is used to perform linear interpolation in a single dimension from a set of data in independent and dependent variables.

This constructor initializes the interpolator from a map containing independent variables as key and dependent variables as value. A look-up scheme can be provided to override the given default.

Parameters
dataMapMap containing independent variables as key and dependent variables as value.
selectedLookupSchemeIdentifier of lookupscheme from enum. This algorithm is used to find the nearest lower data point in the independent variables when requesting interpolation.
boundaryHandlingBoundary handling method, in case the independent variable is outside the specified range.
defaultExtrapolationValuePair of default values to be used for extrapolation, in case of use_default_value or use_default_value_with_warning as methods for boundaryHandling.
