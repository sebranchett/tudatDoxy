11  * Note that the exact implementation of Newton-Raphson as root finder should be updated if
12  * someone would want to use a different root-finding technique.
13  *
14  * By default the eccentricity is used as the iteration procedure. This is because in
15  * optimizing a Cassini-like trajectory, the pericenter radius had about 2-4 NaN values in
16  * 100000 times the gravity assist calculation. The eccentricity iteration had no NaN values
17  * for a similar run in which 100000 gravity assist calculations were done. Also the
18  * eccentricity seemed to require slightly less iterations (does not necessarily mean it is
19  * faster or more accurate).
20  *
21  */
