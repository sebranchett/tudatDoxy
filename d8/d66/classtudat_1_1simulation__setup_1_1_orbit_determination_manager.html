Detailed Description

template<typename ObservationScalarType = double, typename TimeType = double, typename std::enable_if< is_state_scalar_and_time_type< ObservationScalarType, TimeType >::value, int >::type = 0>
class tudat::simulation_setup::OrbitDeterminationManager< ObservationScalarType, TimeType, std::enable_if< is_state_scalar_and_time_type< ObservationScalarType, TimeType >::value, int >::type >

Top-level class for performing orbit determination. All required propagation/estimation settings are provided to this class, which then creates all objects needed for the propagation and estimation process. The parameter estimation itself is performed by providing measurement data and related metadata (as PodInput) to the estimateParameters function.
