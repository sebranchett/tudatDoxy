Detailed Description

template<typename StateScalarType = double, typename TimeType = double>
class tudat::propagators::HybridArcVariationalEquationsSolver< StateScalarType, TimeType >

Class to manage and execute the numerical integration of variational equations of a dynamical system in a combination of single and multiple arcs

In this class, the governing equations are set once, but can be re-integrated for different initial conditions using the same instance of the class.
