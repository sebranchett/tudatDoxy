Detailed Description

Class to define settings for basic tidal gravity field variations, i.e. according to Eq. (6.6) of IERS 2010 conventions.

Parameters
deformingBodiesList of bodies causing tidal deformation
loveNumbersList of Love number for the deformed body. First vector level denotes degree (index 0 = degree 2), second vector level denotes order and must be of maximum size (loveNumbers.size( ) + 2, i.e. maximum degree >= maximum order)
bodyReferenceRadiusReference (typically equatorial) radius of body being deformed
interpolatorSettingsSettings that are to be used to create an interpolator for the gravity field variations immediately upon creation (to be used during propagation). Default is nullptr, in which no interpolation is used, and the model is evaluated during propagation.
