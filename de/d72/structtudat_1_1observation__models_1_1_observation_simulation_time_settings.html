Detailed Description

template<typename TimeType>
struct tudat::observation_models::ObservationSimulationTimeSettings< TimeType >

Base struct for defining times at which observations are to be simulated. Here, only the link end from which the observation is to be calculated is defined. Derived classes are used for defining the times themselves (either directly or through some algorithm).
