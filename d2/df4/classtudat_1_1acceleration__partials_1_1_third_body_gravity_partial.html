Detailed Description

template<typename DirectGravityPartial, typename std::enable_if< is_direct_gravity_partial< DirectGravityPartial >::value, int >::type = 0>
class tudat::acceleration_partials::ThirdBodyGravityPartial< DirectGravityPartial, type >

Class to calculate the partials of a third-body gravitational acceleration w.r.t. parameters and states. This class may be used for any direct gravitational acceleration (central, spherical harmonic, mutual spherical harmonic, etc.), providin a generic third-body partial interface. The template parameter is the derived class of AccelerationPartial for the associated direct acceleration partial.
