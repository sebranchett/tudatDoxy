Detailed Description

template<typename ObservationScalarType = double, typename TimeType = double>
class tudat::observation_models::ObservationSimulatorBase< ObservationScalarType, TimeType >

Virtual base class for the observation simulator class, which is used to compute observable values of a single type of observable This base class is used for practical purposes, as the derived class has a template argument for the observable size, precluding the possibility of making a list of objects for all observation simulators (e.g. one for each observable type)
